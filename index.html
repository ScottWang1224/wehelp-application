第八屆 WeHelp 訓練營申請

1. 個人簡介和申請動機。

我本身是非本科系畢業，自從 ChatGPT 等生成式 AI 爆發後，我開始對開發 AI 應用這個領域產生很大的興趣，也因此開始自學相關技術。再加上這兩年 Vibe Coding 的風潮崛起，我也嘗試使用 AI Agent 輔助開發。

在這個過程中，我確實感受到 AI 能大幅提升開發效率，但也慢慢發現一個問題：當 AI 生成程式碼的速度越來越快，而我 Review Code 的速度逐漸跟不上時，會讓我有一種自己還是「半吊子的開發者」的感覺。

所以我想報名 WeHelp，希望透過更完整、紮實且高強度的訓練，補齊自己的軟體工程基礎，讓自己不只是能靠 AI 工具做出東西，而是能真正理解程式碼、架構設計與工程品質。

2. 曾經做過哪些軟體工程技術相關的學習？若有作品請分享給我們。

我初期是透過彭彭老師的 YouTube 課程學習前後端相關的基本知識，後來也會在 Udemy 上購買自己有興趣的技術課程來自學，例如 Python 後端開發、資料處理與 AI 應用相關內容。

近期我主要在研究 RAG、AI 智能客服相關的應用，並實作了一個健保資料問答系統 Side Project。

健保資料問答系統 GitHub github.com/ScottWang1224/nhi_rag_system 作品相關簡報 Google Drive 簡報連結

3. 如果參與這個訓練，會怎麼安排學習時間？

如果有機會參與訓練，平日會保留主要時段進行課程任務的學習與實作，盡量讓每天都有穩定進度，以跟上訓練目標。

假日則會用來複習當週學習成果、整理筆記，並回頭檢查自己在實作中卡住或理解不夠完整的地方。

4. 請簡述對於軟體工程的領域偏好和職涯想像。

我對後端開發、資料科學領域皆有興趣涉略。相較於單純研究模型本身，我更想學習如何把 LLM API、資料來源、資料庫與後端系統整合起來，打造真正能在實際情境中使用的 AI 應用。

長期來看，我希望自己能成為懂軟體工程基礎，也懂 AI 應用整合的工程師，在產品需求、資料處理與系統開發之間解決實際問題。

5. 請描述一件產生明顯負面情緒的經歷，如何處理該情緒？

我本身比較少有情緒失控的情況，產生明顯負面情緒大多是在與人發生爭執的時候，當下可能會覺得煩躁或不舒服，但我通常不會在情緒最強烈的時候急著反應，通常都是抽空做一些自己有興趣的事就可以快速轉換心情。

6. 關於這份申請網頁，分享一個開發時的技術心得。

我先把問題與答案整理成 Markdown 檔，讓 Codex 讀取後快速寫好一個模板，會比直接從零開始寫更有效率。

7. 如何看待人工智慧的發展對軟體工程師的影響？

「用 AI 寫程式」想必是未來軟體工程師的必備技能，但開發效率快速提升的情況下，若沒有好的軟工基礎，勢必沒有能力維護 AI 產出的 Code。我認為未來軟體工程師會更看重系統架構設計、程式碼品質、需求理解與問題拆解能力。

在資訊爆炸與 AI 快速進步的時代，我覺得保持持續學習的心態也很重要，願意不斷更新自己的工具與觀念，才能跟上技術變化。

9. 其他想要對我們說的事情？

我看過幾屆 WeHelp 畢業學員的成果 Demo，真的能感受到這個訓練的強度與紮實程度，是外面許多一般課程或補習班比較難達到的。也因為看到學員能在幾個月內完成具有完整度的作品，讓我更確定自己希望進入這樣的環境，用更高的標準要求自己成長。
